## What is this project?

This is a personal portfolio website for Arsh Saxena, a web developer and student at VIT Chennai. The portfolio showcases:

- **Web Projects**: Full-featured applications like NPTEL Companion, VIT Chennai Faculty Information, and various utility tools
- **Python Projects**: Interactive games and applications including Snake, Ping Pong, Space Invaders, and more
- **Web Templates**: Downloadable UI components and templates for developers
- **Professional Experience**: Timeline of education, work experience, and achievements
- **Skills & Technologies**: Comprehensive overview of technical capabilities

## Technology stack

### Core framework

- **Next.js 15.5.9**: React framework with App Router
- **React 19.1.0**: Latest React with enhanced capabilities
- **React DOM 19.1.0**: React rendering for web

### Styling & UI

- **Tailwind CSS 3.4**: Utility-first CSS framework
- **Geist Font**: Modern sans-serif font family by Vercel
- **Radix UI**: Accessible component primitives
- **Lucide React & Tabler Icons**: Icon libraries
- **class-variance-authority**: Component variant management

### Animation & interactivity

- **Motion 12.12**: Framer Motion for animations
- **typed.js 2.1**: Typing animation effects
- **next-themes 0.4**: Dark mode implementation
- **react-gradient-animation**: Animated gradient backgrounds
